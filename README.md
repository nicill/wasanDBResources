## Code 

The code used for the experiments described in the paper can be found in this repository: [https://github.com/nicill/kanjiDetection](https://github.com/nicill/kanjiDetection)

However, if you wish to reproduce the experiments or otherwise process the Data and are interested in how the code actually works, we recommend that you have a look at [this page](https://colab.research.google.com/drive/1mjBugZmwa6OSxNaqkrCu4OCslKf2Wy7P?usp=sharing).

In it you will find lots of contents related to Python programming of the materials described in the paper and much more. If you are a beginner in programming, we heartily recommend that you start with the initial courses.
